# SmartHome System

## Описание проекта
Это учебная система управления умным домом. В проекте реализован **центральный контроллер** (`SmartHomeController`) и несколько **умных устройств** (свет, кондиционер, отопление), которые реагируют на события контроллера (смена времени суток, изменение температуры, обнаружение движения).  

Система демонстрирует работу **делегатов, событий и интерфейсов** в C# и позволяет:

- Подписывать устройства на события контроллера.
- Автоматически реагировать на события.
- Управлять устройствами вручную через метод `TriggerDevice`.
- Вести журнал всех событий и действий через `EventLogger`.

## Состав проекта

### 1. `SmartHomeController`
- **События**:
  - `event Action<string> OnDayTimeChanged;`
  - `event Action<int> OnTemperatureChanged;`
  - `event Action OnMotionDetected;`
- **Методы**:
  - `RegisterDevice(ISmartDevice device)` — регистрация устройств.
  - `ChangeDayTime(string timeOfDay)` — смена времени суток.
  - `ChangeTemperature(int temperature)` — изменение температуры.
  - `DetectMotion()` — обнаружение движения.
  - `TriggerDevice(string deviceName, string command)` — ручное управление устройствами.
  - `ShowLog()` — вывод журнала событий.
- Внутренне хранит список зарегистрированных устройств и логирует все действия.

### 2. Классы устройств
Каждое устройство реализует интерфейс `ISmartDevice` и реагирует на события:

#### Light (Свет)
- Подписан на событие `OnDayTimeChanged`.
- Включается утром, выключается ночью.
- Поддерживает команды `On` и `Off`.

#### AirConditioner (Кондиционер)
- Подписан на событие `OnTemperatureChanged`.
- Включается при температуре > 25°C.
- Выключается при температуре < 18°C.
- Поддерживает команды `On` и `Off`.

#### Heater (Отопление)
- Подписан на событие `OnTemperatureChanged`.
- Включается при температуре < 10°C.
- Выключается при температуре ≥ 10°C.
- Поддерживает команды `On` и `Off`.

### 3. EventLogger
- Логирует все события и действия устройств.
- Методы:
  - `Log(string message)` — добавляет запись с отметкой времени.
  - `ShowLog()` — выводит все записи на консоль.

## Требования

## Как запустить
1. Откройте проект в VS Code.
2. Соберите проект:
```bash
dotnet build SmartHomeSystem.csproj
```
3. Запустите программу:
```bash
dotnet run --project SmartHomeSystem.csproj
```
